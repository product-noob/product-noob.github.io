Interactive Tools

A collection of handy web-based tools to assist with everyday tasks. All tools are free to use and work directly in your browser, created using LLMs :)

Calculator

A versatile calculator with standard and scientific functions, history tracking, and memory operations.

QR Code Generator

Generate custom QR codes for URLs, contact information, or any text. Download in multiple formats.

PolyChat

Chat with multiple AI models simultaneously. Compare responses and capabilities across different large language models.

Base64 Encoder/Decoder

Encode text to Base64 or decode Base64 to text. Useful for encoding binary data, URLs, and more.
